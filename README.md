# React Weather App

A modern weather forecast application built with React.js and OpenWeather API.

## Features

- Search weather by city name
- Real-time temperature data
- Humidity and wind speed
- Error handling for invalid cities
- Responsive modern UI

## Tech Stack

- React.js
- Vite
- Axios
- OpenWeather API

## Folder Structure

```bash
src/
├── components/
│   └── WeatherCard.jsx
├── App.jsx
├── App.css
└── main.jsx

```
## Installation
git clone <your-repo-url>
cd react-weather-app
npm install

## Environment Variables
Create a .env file in root:
```
VITE_WEATHER_API_KEY=YOUR_OPENWEATHER_API_KEY
```

## Run Locally
```
npm run dev
```

## API Used
OpenWeatherMap API
